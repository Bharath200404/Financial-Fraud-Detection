Intelligent E-Commerce Fraud Detection & Risk Analytics Platform

The Intelligent E-Commerce Fraud Detection & Risk Analytics Platform is an end-to-end machine learning web application designed to identify suspicious e-commerce transactions using behavioral analytics and fraud risk prediction techniques.

The system analyzes:
- transaction amount
- payment method
- account age
- transaction timing
- device usage
- address mismatch patterns
 to identify potentially fraudulent activities and generate fraud risk scores.

🚀 Features
- Fraud Probability Prediction
- Real-Time Risk Scoring
- Fraud Analytics Dashboard
- Interactive Streamlit Web App
- Feature Engineering & Behavioral Analysis
- Threshold-Tuned XGBoost Model

🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
- XGBoost
- Streamlit
- Plotly

📊 Key Insights
- Late-night transactions showed higher fraud probability.
- Newly created accounts were more vulnerable to fraud.
- High-value transactions exhibited elevated risk.
- Address mismatches indicated suspicious activity patterns.

🤖 Machine Learning Workflow
- Data Cleaning & EDA
- Feature Engineering
- Model Comparison
- Threshold Tuning
- Fraud Risk Prediction

Models Used:
- Logistic Regression
- Random Forest
- XGBoost
